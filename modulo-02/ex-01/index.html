exemplo de cores RGB

O sistema RGB (Red, Green, Blue) define uma cor baseando-se na intensidade dessas três cores primárias de luz. O valor varia de 0 a 255 para cada canal. É a forma mais comum de pensar em cores para telas digitais.

Como usar: Utilizamos style="background-color: rgb(255, 0, 0);" (neste exemplo, o resultado seria um vermelho puro).

exemplo de cores HSL

O HSL (Hue, Saturation, Lightness) é mais intuitivo para humanos. Ele define a cor através da Matiz (o ângulo na roda de cores), Saturação (vivacidade) e Luminosidade (brilho).

  • Hue (Matiz): 0 a 360 (representando os graus no círculo cromático).
  • Saturation: 0% a 100%.
  • Lightness: 0% (preto) a 100% (branco).

Como usar: style="background-color: hsl(240, 100%, 50%);" (resulta em um azul vibrante).

exemplo de cores HEXADECIMAL

A representação Hexadecimal é a favorita de muitos desenvolvedores por ser compacta. Ela usa uma combinação de números (0-9) e letras (A-F) para definir os níveis de RGB, sempre precedida pelo caractere #.

Como usar: style="background-color: #00FF00;" (resulta em um verde puro). Os dois primeiros dígitos são para o Vermelho, os dois centrais para o Verde e os dois últimos para o Azul.

exemplo de cores Cores Com Transparência

Para adicionar transparência às cores, utilizamos variações dos modelos acima, adicionando o canal Alpha (que varia de 0.0 totalmente transparente a 1.0 totalmente opaco):

  • RGBA: rgba(255, 0, 0, 0.5); (Vermelho com 50% de transparência).
  • HSLA: hsla(120, 100%, 50%, 0.3); (Verde com 30% de transparência).
  • HEX com 8 dígitos: #FF000080; (Onde os dois últimos dígitos representam a opacidade).
