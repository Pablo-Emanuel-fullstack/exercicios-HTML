Minhas Principais Dificuldades na Execução Deste Projeto

Para abordar esse tema, vou usar uma linguagem que fique bem simples para minha compreensão em consultas futuras.

O que mais me travou neste módulo?

Sendo bem direto, o que mais me travou foi o iframe. Por quê? Sinceramente, não sei (hehehe). Só fiz o que o Guanabara fez no vídeo Vídeo em Questão e segue o baile. Acredito que seja por falta de prática, mas eu vou fazer alguns exercícios para praticar isso.

Explicando o que eu aprendi sobre o iframe

O iframe basicamente é uma forma mais leve de trazer vídeos, imagens, páginas, etc., para a nossa página. Isso é maravilhoso até certo ponto. É bom porque diminui o peso do nosso site, fazendo-o ficar mais leve e rodando mais rápido na máquina do usuário.

O que o torna uma coisa que trava alguns desenvolvedores é a dificuldade de manipular e posicionar da maneira correta, ou seja, a maneira de tratar o iframe depois de coletado do site em questão.

Como eu fiz neste projeto

Primeiro, criei uma div com a classe video para colocar meu iframe. Depois, já no meu CSS, eu chamei essa classe e passei os seguintes parâmetros: 

                    div#video{
                    background-color: var(--cor6);
                    padding: 20px;
                    padding-bottom: 56.25%;
                    margin: 20px -20px 20px -20px;
                    position: relative;
                    }

Essa parte em específico é muito importante para o nosso iframe. Após isso, chamei de novo nossa classe, desta vez apontando para o nosso elemento: div.video > iframe.

Agora eu passei outros atributos, desta vez de posicionamento: position: absolute;. Quando adicionar, vai parecer que você quebrou o código! Calma, não se desespere. Assim que você começar a passar as posições (top, left, height, width), o código vai se consertando. 

                    div#video>iframe{
                    position: absolute;
                    left: 5%;
                    height: 90%;
                    top: 5%;
                    width: 90%;
                    }
